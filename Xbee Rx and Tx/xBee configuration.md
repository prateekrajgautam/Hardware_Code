# Firmware: Zigbee

#### BS/Coordinator/Master/Tx

1. update firmware to Zigbee

2. change these setting

   | keyword | Value                        |
   | ------- | ---------------------------- |
   | ID      | 1234                         |
   | DL      | 0                            |
   | DF      | FFFF (for BROADCAST)         |
   | CE      | 1(Enable)                    |
   | JV      | 0 (Disabled)                 |
   | NI      | COORDINATOR (name)           |
   | AP      | 0 (Transparent) for API mode |
   |         |                              |
   |         |                              |
